Mike's HTML & CSS Cheatsheet

Basic HTML Page Layout

Tag/Syntax Description
<head> </head> Sets up part of site that contains meta-data, links to external style sheets, and the page title.
<header> </header> Creates an introductory section to coming content. Typically place for navigational aid.
<body> </body> Contins the main body elemnts of the webpage.
<footer> </footer> Area at bottom of the webpage. Good place for placing author, copyright and contact info.

HTML Commands

Tag/Syntax Example Description
<h1> Header </h1> Establishes a heading. Can be h1 through h6.
<p> text </p> Establishes a paragraph. There is an automatic line break after the closing tag.
<br> Creates a line break. This is a self-closing tag.
<img -link to image- > Displays an imagee on the webpage.
<form> </form> Sets up interface to collect and submit user supplied information.
<input type='text' name='username' /> Creates an interactive element allowing user to make input. Can use 'radio', 'checkbox', 'text' for example. Y0u must include a "type= "and a "name= ". It has no closing tag.
<em></em> Emphasizes text, browsers usually render as italics.
<ol></ol> Creates an orderd list. List items will be numbered.
<ul></ul> Creates an unordered list. Items will have bullets.
<li>element</li> The line tag is used to add items to a list. It is used in both ordered and unorderd lists.
<span attributes= element><span> This is used to group text or elements together for styling purposes.
<a>href="url">'text desciption of link'</a> This is the 'anchor' element that creates either an internal or external link.

Common CSS Rules

Specifity: ID - Class - Type

Property Name: Value Affect
font-family: Monoscape, "Times New Roman, serif; This will change the targeted font. It will attempt the fonts in order. Multiple fonts are listed in case the browser doesn't support the selected font.
font-size: 30px; Changes the targed item's font to whatver value is after the colon.In this case 30px.
color:red; Changes the font color to red.
background-color: blue Makes the background of the selected area the color selected.
font-weight: bold Sets how bold the text is. It can be light, normal, bold, etc. Or, it can be a number between 100-900 for extra control.
text-transform: value Transforms the font to the given value. Values are none, uppercase, lowercase, capitalize, full-width.
text-shadow: value Applies up to 4 proterties to the font: horizontal offsets, vertical offset, blur radius, base color. You can create multiple shadows by seperating another set of values by a comma.
text-align: Allows you to justify the text left, right, center, or justify.
Important! You must be sure to link the HTML and CSS file!! The syntax is: <link rel="stylesheet" href="styles.css"/> It goes in your head section.
Don't forget: You can stack both selectors and rules in the same command.
